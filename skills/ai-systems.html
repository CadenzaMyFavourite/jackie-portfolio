AI Systems & Tooling

Descriptions of AI-related technologies I’ve worked with, including how they fit into real workflows.

LLMs (OpenAI, Azure)

Large language models power natural language understanding and generation. I use them for chatbots, summarization, and automation. Workflow: send prompts via API, handle token limits, manage context, and fine‑tune when needed.

LangChain / LangGraph

These libraries help orchestrate LLM calls, memory, and chains of reasoning. In practice I compose agents that call multiple models, fetch data, and post-process results.

RAG pipelines & agents

Retrieval‑augmented generation adds a search step before querying the model. I integrate vector databases and embedding services to provide up‑to‑date context for LLM prompts.

Prompt engineering

Crafting effective prompts is key to getting useful outputs. I iterate by writing instruction templates, testing edge cases, and tuning temperature/parameters.
