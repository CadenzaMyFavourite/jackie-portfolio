Programming Languages & Concepts

This page covers the core programming languages I work with and how they are applied in real projects.

Python (Advanced)

Python is my go-to language for rapid prototyping, data processing, scripting, and backend services. In production I use it for web APIs (FastAPI/Flask), automation pipelines, and machine learning tasks.

Typical workflow: write code in modules, use virtual environments, write unit tests with pytest, and deploy via Docker/CI.

TypeScript

TypeScript adds static typing to JavaScript and is used when building web frontends or serverless functions. It helps catch errors early and improves code maintainability.

Java

Java is valuable for backend systems requiring strong typing and performance. I’ve used it for designing object‑oriented services and interfacing with relational databases.

C++

C++ skills come in handy for systems programming, algorithm contests, or performance‑sensitive modules. I use it for custom data structures and low‑level tasks.

SQL

SQL is essential for querying and manipulating relational data. I write complex joins, optimize queries, and design normalized schemas to support applications.
