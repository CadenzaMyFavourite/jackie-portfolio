Backend Systems & Data Infrastructure

Tools and practices I use to build scalable services and manage data.

FastAPI, Flask, REST API design

I develop web services with Python frameworks such as FastAPI and Flask. Design focuses on clear endpoint contracts, validation, and documentation (OpenAPI).

Async programming & service architecture

Async patterns (async/await) let services handle many requests concurrently. I also structure applications into layers (handlers, services, repositories) for maintainability.

PostgreSQL, schema design, query optimization

Relational databases are the backbone of many applications. I create normalized schemas, write indexes, and analyze query plans to keep performance high.

Vector DBs (Chroma, Pinecone), embedding pipelines

For semantic search and RAG systems I use vector databases. The workflow involves generating embeddings from text, storing them, and retrieving nearest neighbors during query time.
